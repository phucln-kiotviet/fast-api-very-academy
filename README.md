## Install package

- Create `venv`:

```
python3 -m venv venv
```

- Install package:
```
pip3 install -r requirements.txt
```
- Package list:
    - `fastapi`
    - `sqlalchemy`
    - `pydantic`
    - `alembic`
    - `psycopg2-binary`
    - `uvicorn`
